## 项目架构

- **数据层**：PostgreSQL + DuckDB（OLAP）
- **ETL**：Python (Pandas) + dbt
- **API**：FastAPI / Django REST Framework
- **可视化**：Apache Superset + ECharts
- **AI 增强**：OpenAI Function Calling
- **部署**：Debian 13 + Docker + Nginx
